Tic Tac Toe

Tic-tac-toe is a game in which two players take turns in drawing either an ' O' or an ' X' in one square of a grid consisting of nine squares. The winner is the first player to get three of the same symbols in a row. To make the game a little more interesting we use company logos instead of an 'O' or an 'X' 😜

To start the game please select a company you want to play against and click on "Let the game begin"
